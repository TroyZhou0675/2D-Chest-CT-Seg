# 胸部器官 2D CT 分割项目

[English](./README.md) | [简体中文]

该项目专注于 2D CT 胸部器官分割任务。支持使用原始图像和掩码（Masks）从零开始构建和训练深度学习模型。

## 核心特性

- **从零训练**: 完整的训练流程，支持从 Scratch 开始训练模型。
- **数据处理**: 
  - 输入：256*256 灰度图像。
  - 自动转换：内置转换程序，可将 **RGB 彩色掩码图像** 自动转换为模型训练所需的 **Label Map（标签图）**。
- **模型库**: 提供三种主流模型选择：
  - `U-Net`
  - `Simple_NestNet` (即 UNet++)
  - `NestNet with Backbone` (带骨干网络的 NestNet)

## 快速上手

1. **准备数据**: 将 256*256 的灰度图和对应的 RGB 彩色掩码放入数据目录。
2. **预处理**: 运行内置程序将 RGB 掩码转换为 Label Map。
3. **训练**: 选择合适的模型（U-Net / NestNet 等）并启动训练脚本。
